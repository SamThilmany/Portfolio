The hourly wage applies to all work done outside the website packages. For example, the addition of new functions, updating the content or finding and fixing bugs in existing websites, the design of a business card, and such like. If you want to hire another web developer, you can also ask me to just create a website mockup.

I also offer WordPress training, where I show you further features, and some neat tricks which are useful in your daily work with WordPress (Screencast).

The Static Website package includes the creation of a complete website that is created according to your wishes. In addition, you receive a year of free web hosting on my webspace, including a .de domain. Should you want new business cards that match the new design of the website, you will also get these for free too (only the design!).

After the first year, you pay an annual fee for the webspace, your domain and the features used, such as email addresses, databases and backups.

The Custom WordPress Website package includes all the features of the Static Website package with the particularity that this is a more complex system that allows you to edit the contents by yourself.

If you have never used WordPress before, I will teach you the very basics, so that you feel comfortable to edit the contents. This training consists in a Screencast.
